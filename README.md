# Weather App

An app which shows the weather and Map of a searched city and on the basis of current location.



## Live
[Weather app](https://sensational-pastelito-9be952.netlify.app/)




## Features

- Current Weather
- Detect current location
- City Map
- 6 days Weather forecast
